🔗 Coerencia Textual

Coerencia: quando o texto faz sentido, tem logica e unidade de ideias. Um texto coerente tem tema central, progressao de ideias e sem contradicoes.

Sem coerencia (texto ruim):

"Fui a obra. O ceu e azul. A viga e de concreto. Gosto de futebol."

Problema: frases sem conexao logica entre si

Com coerencia (texto bom):

"Fui a obra verificar as vigas. As vigas de concreto armado estavam em boas condicoes. O cronograma sera mantido."

Correto: cada frase avanca o mesmo tema com logica

O que destroi a coerencia:
Contradicao Fuga do tema Ideias desconexas Informacoes falsas
Exercicio: leia um paragrafo e pergunte: "isso faz sentido? cada frase conecta com a anterior?"

🔄 Coesao Textual

Coesao: os mecanismos linguisticos que LIGAM as frases entre si — as "pontes" do texto. Diferente da coerencia (sentido geral), coesao sao as palavras conectoras.

TipoDescricaoExemplos
ReferencialUsa pronome ou sinonimo para nao repetir"A viga... ela... o elemento estrutural..."
SequencialUsa conectivos para ordenar"primeiro... depois... por fim..."
Conectivos mais usados:
FuncaoConectivos
Adicaoe, tambem, alem disso, ainda
Oposicaomas, porem, entretanto, no entanto
Causaporque, pois, ja que, uma vez que
Conclusaoportanto, logo, assim, dessa forma
Tempoquando, enquanto, depois, antes, em seguida
Exemplo com coesao:

"O concreto foi lancado. Em seguida, a equipe fez o nivelamento. Por fim, cobriu com lona para cura."

⚖️ Principios da Coerencia e Coesao
PrincipioO que significa
Nao contradicaoUm texto nao pode afirmar e negar a mesma coisa
Nao tautologiaNao repetir a mesma ideia com palavras diferentes sem acrescentar nada
RelevanciaCada parte do texto deve contribuir para o tema central
ProgressaoO texto deve avancar, trazer informacoes novas a cada paragrafo
Na pratica em Engenharia: Um relatorio tecnico deve ter:
1. Introducao — o que foi feito
2. Desenvolvimento — como foi feito, com dados
3. Conclusao — resultado e recomendacoes
Sem desviar do tema em nenhum momento.

🔢 Media, Mediana e Moda
Marque 🔢 — use a calculadora para todos os calculos desta secao.
MEDIA (x̄)
x̄ = Σxi / n    (soma de todos ÷ quantidade)
Alturas (m): 1,70; 1,65; 1,80; 1,75; 1,68
Soma todos1,70+1,65+1,80+1,75+1,68 = 8,58
Divida por n=58,58 ÷ 5 = 1,716 m
MEDIANA (Md)

Valor do meio apos ordenar os dados em ordem crescente.

CasoComo calcularExemplo
n impar (5)Valor central1,65; 1,68; 1,70; 1,75; 1,80 → Md = 1,70
n par (6)Media dos dois centrais(1,70+1,75) ÷ 2 = 1,725
MODA (Mo)

Valor que aparece mais vezes nos dados.

Dados: 20, 20, 25, 30, 20, 25Mo = 20 (aparece 3 vezes)
Bimodal (2 modas)Dois valores com mesma frequencia maxima
Sem modaTodos aparecem igual numero de vezes

📐 Variancia e Desvio Padrao 🔢
Variancia (s²) = Σ(xi - x̄)² / (n−1)
Desvio Padrao (s) = √s²
Exemplo: resistencia do concreto (MPa): 28, 30, 32, 25, 30
1. Calcule a media(28+30+32+25+30)/5 = 145/5 = 29 MPa
2. (28-29)² = (-1)²= 1
2. (30-29)² = (1)²= 1
2. (32-29)² = (3)²= 9
2. (25-29)² = (-4)²= 16
2. (30-29)² = (1)²= 1
3. Soma dos quadrados1+1+9+16+1 = 28
4. Divida por (n-1)=428/4 = 7 (variancia)
5. Raiz quadrada de 7√7 ≈ 2,65 MPa (desvio padrao)
🔢 Use a calculadora cientifica para as potencias (x²) e raiz quadrada (√).

📊 Histogramas 🔢

Grafico de barras para variaveis continuas, sem espaco entre as barras. Eixo X: classes (intervalos). Eixo Y: frequencia.

Como construir:
1
Defina o numero de classes: k ≈ √n (raiz de n arredondada) 🔢
2
Calcule a amplitude total: R = maior valor - menor valor
3
Calcule o intervalo de classe: h = R / k 🔢
4
Monte a tabela de frequencias por classe
5
Desenhe as barras sem espaco entre elas

🔁 Loops — while e for
WHILE — enquanto condicao for verdadeira:

    contador += 1

    print(f"Estudar: {materia}")
RANGE — gera sequencia de numeros:
range(5)          # 0, 1, 2, 3, 4
range(1, 6)       # 1, 2, 3, 4, 5
range(0, 10, 2)   # 0, 2, 4, 6, 8 (de 2 em 2)

for i in range(1, 31):
    print(f"Dia {i} de junho")
BREAK e CONTINUE:
# break: para o loop completamente
for i in range(10):
    if i == 5:
        break
    print(i)      # imprime 0, 1, 2, 3, 4

# continue: pula para a proxima iteracao
for i in range(10):
    if i % 2 == 0:
        continue   # pula pares
    print(i)      # imprime so impares: 1, 3, 5, 7, 9
